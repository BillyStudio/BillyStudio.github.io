with关键字

with语句在Python中可以实现上下文管理协议。写法较try-catch-finally更加简洁。

with语句定义了一个上下文. 执行 with 语句时, 首先调用上下文对象 context 的 enter(), 其返回值赋值给 var; 离开 with 语句块时, 最后调用 context 的 exit(), 确保释放资源.
