算法题中经常会遇到的字符串处理题.

最长回文子串

问题描述

当一个字符串和他的逆序串完全相等时，我们称这是一个回文串(Palindrome)，如 “malayalam”, “dad”, “appa”等。
输入一个字符串，输出其最长回文子串的长度
来源：POJ 3974，SPOJ-IPS

奇数串与偶数串的处理

如果采取的是逆序之后比较的办法, 奇数串和偶数串没有差异. 逆序后比较的方法复杂度比较高, 故一般是从中间往两边扩散的方法来进行判定. 注意从中间开始往两边扩散的方法判定时奇数串和偶数串的判定标准是不一样的. 对于一个奇数回文串, 比如 madam, 除去中间的 d 字符, 剩下的两两匹配. 对于一个偶数回文串, 如 elle, 都是两两匹配的.

第一种办法最直接，就是将奇数串与偶数串使用两段语句，分开来处理。

第二种办法是加入特殊字符进行填充，将奇数串和偶数串都转化为奇数串。

第三种办法利用到了一个特性，也就是偶数串最中间的两个字母一定是相同的，所以我们在循环的过程中判断相邻两个字符串是否相等，来统一识别出奇数串和偶数串。

另一种办法叫做Manchester算法，这个算法有一种动态规划的思想在里面，充分利用先前计算得到的结果。具体阐述为：
如果最靠右的回文子串右边界为right，中心为id，如果 i 在 id 和 right 之间，其最大子串半径 rad[i] >= min(rad[2*id-i], right-i)；如果i在right右边则无法判断，只能暂时初始化为1。进一步，i 及小于 i 的 rad 值一旦确定后，就可以将 i 作为新的最靠右的回文子串中心，即 id=i.

最长公共子序列（Longest Common Sequence）和最长公共子串（Longest Common Substring）

两个都可简称为LCS，最长公共子串和最长公共子序列都可以用动态规划解决。所不同的是，当串 s1[i]≠s2[j],i>0,j>0 时，公共子串的 dp[i][j]=0 而公共子序列的 dp[i][j]=max{dp[i-1][j],dp[i][j-1]}，最长公共子串需要用一个额外String变量存储，而最长公共子序列就是 dp[s1.len-1][s2.len-1]。下面是两个问题的动态规划算法实现
