数据结构与算法笔记 —— Dijkstra 算法与 C++ 优先队列

Posted on 2018-07-18

Dijkstra 算法是解决单源最短路问题的经典算法.

最短路问题

在一个连通图中, 如何求解从一个节点到另一个节点的最短路径, 称为最短路问题.

Dijkstra 算法

使用一个 dist 数组储存从原点到该点的最短距离, 原点初始化为0, 其余全部初始化为不可达. 首先原点和最短距离作为一个 pair 加入优先队列, 优先队列按照最短距离的大小逆序排列. 从优先队列中依次取出最短边, 考虑最短边对应的结点的周围节点, 若通过此节点到达某周围节点的路径更短, 则进行更新. 这里需要注意的是需要判断是否出现重复路径, 判断条件可以是优先队列弹出的最短边不能大于dist数组中的最短边大小.

priority_queue

C++ 中的优先队列是一个容器, 插入时为对数时间的复杂度, 读取最大值或最小值为常数时间的复杂度.

使用 STL 中的 std::priority_queue 需要包含头文件 queue.

默认是从大到小排序, 也可以通过指定 greater<T> 参数进行从小到大排序. 如果自定义比较方法的一定要注意, 这里需要定义一个比较 type 而不是比较函数. 在用于比较的类型(struct或class)中重写 operator() 而不是 operator<.
