交换排序

交换排序是一种借助记录交换进行排序的方法。
比较著名的有冒泡排序和快速排序。

冒泡排序

i 从 0 开始遍历到 n，相邻两个元素 r[i] 和 r[i+1] 进行比较, 将较大的元素交换到后面, 这样经过一次 i 的遍历之后, 最大的元素就会沉底. 然后 i 从 0 开始遍历到 n-1, 同样的方法, 第二大的元素就会沉底. 依次类推. 时间复杂度 O(n^2)

快速排序

经典的快速排序方法首先选定一个数 (一般为第一个元素) 作为基准(Pivot), 接下来通过 key 的逆序比较(用 j 遍历)和正序比较 (用 i 遍历) 找出倒着数第一个比基准小的和正数第一个比基准大的数, 然后把比基准小放到 i 的位置, 比基准大的放到 j 的位置, 当 i == j 时, 就找到了基准所在的位置。假设数组arr从下标l到下标r进行一次遍历，返回pivot所插入的位置，双指针遍历的代码描述如下：(assert arr.length >= r+1)

分成了基准位置前和基准位置后两块, 再对其递归排序. 时间复杂度 O(nlog_2 n)

选择排序

选择排序属于稳定排序(稳定排序通俗地讲就是能保证排序前2个相等的数其在序列的前后位置顺序和排序后它们两个的前后位置顺序相同。), 是最符合人们思维习惯的一种排序方法. 也是考试中的常见出题范围.
排序的基本思想是先选出最小 (或最大) 的元素, 然后在剩余的元素中选出最小 (或大) 的元素作为第二个排序好的元素, 以此类推. 选择排序目前主要分为两大类, 即直接选择排序和堆排序.

直接选择排序

先从头开始遍历, 记录最小元素对应的序号 j, 遍历结束后边将第一个元素和j对应元素互换, 这样就将最小的元素放到了第一个位置. 然后从第二个元素开始遍历, 类似地就能找出第二小的元素. 这样经过 n-1 次遍历, 排序结束, 时间复杂度 O(n^2), 属于稳定排序.

堆排序

如果一棵完全二叉树的每个子树, 根节点都是最大的, 那么称为大根堆, 如果根节点都是最小的, 那么称为小根堆。
完全二叉树可以用数组表示，所以堆排序虽然输入的是数组，但其逻辑结构是完全二叉树。
大根堆用于升序排列，小根堆用于降序排列。
