安装macOS虚拟机之后要做的几件事

Posted on 2018-06-16

高大上的mac电脑不是我等穷屌丝用得起的, 但是macOS系统还是可以在虚拟机里玩一玩的

虚拟机系统配置

安装VMware Tool

设置共享文件夹

操作系统配置

操作系统安全

删除其他用户的登录选项

sudo defaults write /Library/Preferences/com.apple.loginwindow SHOWOTHERUSERS_MANAGED -bool FALSE

系统优化与美化

加快退格键和delete键的速度

进入 System Preferences > Keyboard > Key Repeat, 调到最快.

编程环境配置

安装Xcode

从AppStore安装即可, 前提是要有一个Apple ID

安装brew包管理器

进入homebrew的官网, 从官网上粘贴安装命令到终端中.

安装Vim

下载最新的VIm并替换系统自带的Vim (系统自带的功能不全)

安装完毕后使用vim编辑编辑文本文件. 如果使用vi, 仍然是系统原来自带的vim
