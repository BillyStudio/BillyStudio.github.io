二、操作系统配置

2.1 键盘与输入法

设置双拼自然码方案

首先在设置中添加简体双拼输入法，然后打开终端输入

即可

设置Shift切换输入法

安装 Karabiner-Elements

配置文件~/.config/karabiner/assets/complex_modifications/shift_toggle_chinese_english.json：

取消大写锁定切换输入法：

设置字母按键重复

普通设置里设置按键重复，对于space、delete等都是可以支持的，但对于一些字母按键长按后会显示带accent的字母形式（如àáâäæ），对于平时只需要输入中英文的需求来说完全可以disable掉，让输入更加顺畅

2.2 鼠标

鼠标滚轮与触控板滚动方向相反

下载Scroll Reverser

操作系统安全

删除其他用户的登录选项

sudo defaults write /Library/Preferences/com.apple.loginwindow SHOWOTHERUSERS_MANAGED -bool FALSE

系统优化与美化

加快退格键和delete键的速度

进入 System Preferences > Keyboard > Key Repeat, 调到最快.

三、编程环境配置

安装Xcode

从AppStore安装即可, 前提是要有一个Apple ID

安装brew包管理器

参见brew镜像助手和阿里源Homebrew镜像

安装Vim

下载最新的VIm并替换系统自带的Vim (若系统自带的功能不全)

安装完毕后使用vim编辑编辑文本文件. 如果使用vi, 仍然是系统原来自带的vim
