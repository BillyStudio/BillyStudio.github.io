Transfering Hexo blog files to another computer

Posted on 2019-07-07 Edited on 2019-08-18
Symbols count in article: 409 Reading time ≈ 1 mins.

With the time passing, we might face the issue of transfering our blog files when we change our writing computer.

My solution is direct and simple. First install hexo. Only hexo, no need for those modules or hexo themes.

Then just copy the blog files except .git and .git_deploy files to your new computer. Test its validity by hexo s. If it is ok, then rename this directory and git clone the blog file from github. Then move those all files back into it. Finally test by hexo deploy -g. Thatâ€™s all.
