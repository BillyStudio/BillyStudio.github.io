Likewise, we can write an algorithm for it:

Given x, y, f(x,y) = x*x + y*y + 1
do:
    fx := 2 x + 1
    fy := 2 y + 1
    tmp := sqrt(fx^2 + fy^2)
    cos_a = fx / tmp
    sin_a = fy / tmp
    x := x - s cos_a
    y := y - s sin_a
while (f is decreasing)
